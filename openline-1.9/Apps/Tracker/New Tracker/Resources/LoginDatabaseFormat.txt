HEADER- 2 bytes - Database Version (Value = 1) (Big Endian)- 4 bytes - Items Count (Big Endian)ITEMS- 1 byte - Active/Inactive (Value=1 -> Active, Value=0 -> Inactive)- 32 bytes - Login (pstring - first byte = string size, 		    starting with second byte = string data)- 32 bytes - Password (pstring - first byte = string size,                        starting with second byte = string data)- 33 bytes - Date (pstring - first byte = string size,                    starting with second byte = string data) 	     	  (Format - MM/DD/YYYY HH:MM:SS AM/PM)                   (Example - 10/01/1999 12:01:14 PM)- 6 bytes - Reserved (Value = 0)
